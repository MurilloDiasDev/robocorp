# robocorp

RPA Challenge 2.0

Automate the process of extracting data from a news site.

The work item must be created as example json below
{
    "phrase": "dollar",
    "section": "tech",
    "months": 2
  }

Possible sections or categories to search:

    astrology
    business
    games
    entertainment
    lifestyle
    media
    opinion
    page six
    personal finance
    photos
    preview
    promotions
    real estate
    shopping
    sports
    sports video
    tech
    us news
    video
    world news
